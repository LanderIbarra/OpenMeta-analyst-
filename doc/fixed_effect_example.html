To perform this analysis in Open Meta-Analyst, do the following steps:

  1. Open the example data file BCG.oma in the sample_data directory.
  2. Select meta-analysis on the Analysis menu.
  3. In the Method & Parameters dialog, select Binary Fixed Inverse Variance as the analysis method.
  4. Click OK.
