Opening an Existing Data Set - amino.oma

To open an existing data set, do the following steps:

  1. Select open from the File menu.
  2. Select the file amino.oma. All Open Meta-Analyst data sets have the extension "oma."
  3. Click Open.

Open Meta-Analyst displays the data shown below.
