Diagnostic-Only Methods

  • HSROC - Hierarchical standard receiver operating cuve

HSROC - Hierarchical standard receiver operating curve

The HSROC method performs a joint meta-analysis of sensitivity and specificity of diagnostic test data. Estimation is carried out using a Bayesian approach, implemented via a Gibbs sampler. See Analysis Options for a description of the available options for the method.
