Running the Analysis

To perform a meta-analysis on the current data set, select the Analysis menu, as shown below.

To perform a basic analysis, select meta-analysis. This opens the Methods & Parameters dialog, as shown below.

You can select a method for the analysis in the analysis method field. The default method is Binary Random, which uses the DerSimonian-Laird method to fit a random-effects model. See Analysis Options for a complete description of the available methods and options shown in the dialog.

Click OK to perform the analysis. The following results are displayed in the results/analysis window, as shown below:

  • The Summary contains the overall numerical results of the meta-analysis, including the summary estimate, confidence interval, and heterogeneity statistics.
  • The Forest Plot provides a graphical display of the point estimates and confidence intervals for each study, as well as the overall estimate.
