Entering Data

You can enter data into a new data set in either of the following ways:

  • Copy and paste data from an existing spreadsheet, such as an Excel spreadheet.
  • Type the data directly into the fields in the Open Meta-Analyst window.

Copying and Pasting Data

To copy and paste data:

  1. Select a set of columns in the spreadsheet, corresponding to a contiguous set of columns in Open Meta-Analyst. Do not include column labels.
  2. Right-click the column in Open Meta-Analyst corresponding to the first column of data you selected in the spreadsheet.
  3. Select paste in the drop-down menu.

Note: If there is a missing column in the spreadsheet data - for example, if the study year is not present - add an empty column, corresponding to the missing column, before copying.

Entering Data Directly

To enter data directly into open Meta-Analyst:

  1. Click the study name field and type a name for the first study. As an option, you can also add the year of the study.
  2. Enter data in the fields to the right of "year".
