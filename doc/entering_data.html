Entering Data

You can enter data into a new data set in either of the following ways:

  • Enter the data directly into the fields in the Open Meta-Analyst window.
  • Copy and paste data from an existing spreadsheet, such as an Excel spreadheet.

To enter data directly into open Meta-Analyst:

  1. Click the study name field and type a name for the first study. As an option, you can also enter the year of the study. When you click anywhere outside the study name field, a new row is added to the table for the next study.
  2. Enter data in the fields to the right of "year". For a binary data set with two-arm studies, the following fields are displayed:
    • tx An - Number of events in the treatment group
    • tx AN - Number of subjects in the treatment group
    • tx Bn - Number of events in the control group
    • tx BN - Number of subjects in the control group

The columns shaded yellow, which are automatically calculated from the data, display the effect sizes and bounds for the confidence intervals:

  • upper - The upper bound of the confidence interval
