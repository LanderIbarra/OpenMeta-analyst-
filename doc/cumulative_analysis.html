Cumulative Meta-Analysis

This section shows how to perform a cumlative meta-analysis. The example uses the data set amino.oma, which you can open by selecting open on the File menu.

To run the cumulative meta-analysis,

  1. Select cumulative meta-analysis from the Analysis menu.
  2. In the Method and Parameters dialog, select the meta-analysis method and parameters.
  3. Click OK.
