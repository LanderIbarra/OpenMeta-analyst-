The following analysis options are available in the Methods & Parameters dialog.

Common Options

  • conf.level - The confidence level for the estimate.

  • digits - The number of digits after the decimal point to be displayed in results.

Fixed-Effect Models

  • adjust - Number added to the data for tx An, tx AN, tx Bn, and tx BN, according to the the value of the to parameter.

  • to - Can take one of the following values:
    • only0 - The value of adjust is only added to the data when at least one of tx An, tx AN, tx Bn, and tx BN equals 0 or NA.

    • all - The value of adjust is always add to the data.

Random-Effect Models

  • rm.method - Specifies the method for the model.
