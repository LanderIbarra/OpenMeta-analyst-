Forest Plot Options

To set options for forest plots, select one of the analysis methods from the Analysis menu. Then, in the Methods & Parameters dialog, select the forest plot tab.

 

In the forest plot pane, you can specify labels for each of the columns. To prevent a column from being displayed in the forest plot, clear the show check box. You can also specify the following options:

  • x label - the label on the x-axis
  • x ticks -
  • save image to: - The local directory in which the forest plot is saved.
