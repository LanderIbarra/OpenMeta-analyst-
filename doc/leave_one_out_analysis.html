Leave-One-Out Meta-Analysis

In leave-one-out meta-analysis, you perform a meta analysis on each subset of the studies obtained by leaving out one study. This enables you to see how removing each individual study affects the overall estimate.

This section provides an example that shows how to perform a cumlative meta-analysis. To run the example, do the following steps:

  1. Select Open on the File menu and open the file amino.oma.
  2. Select cumulative meta-analysis from the Analysis menu.
  3. In the Method and Parameters dialog, select the meta-analysis method and parameters.
  4. Click OK.
