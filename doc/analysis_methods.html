Meta-Analysis Methods

This section describes the analysis methods available in Open Meta-Analyst.

Binary Methods

  • Fixed-effect inverse variance
  • Fixed-effect Peto
  • Mantel-Haenszel
  • Random-effects

Continuous Methods

  • Fixed-effect inverse variance
  • Random-effects

Diagnostic Methods

  • Fixed-effect inverse variance
  • Random-effects

Random-effects methods include the following:

  • HE - Hedges estimator
  • DL - DerSimonian-Laird
  • SJ - Sidik-Jonkman
  • ML - Maximum likelihood
  • REML - Restricted maximum likelihood
  • EB - Empirical Bayes
