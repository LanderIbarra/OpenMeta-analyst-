The Open Meta-Analyst Window

To open a data set in the main Open Meta-Analyst window, do the following steps:

  1. Select open in the File menu.
  2. In the Open File dialog, select the file you want to open.
  3. Click Open.

The figure below shows the sample data set amino.oma in the main window.

The window displays the raw data for the studies in the data set, as well as the effect sizes and confidence intervals, which are calculated automatically from the raw data. In this example the data is binary, and the raw data columns display the following:

  • tx An - Number of events in the first treatment group
  • tx AN - Number of subjects in the first treatment group
  • tx Bn - Number of events in the second treatment group
  • tx BN - Number of subjects in the second treatment group

Besides binary data, you can also use Open Meta-Analyst to perform analyses on continuous or diagnostic data. For these data types, different values are displayed in the raw data columns - see Data Types.

The yellow columns, which are automatically calculated from the raw data, display the effect sizes and confidence intervals:

  • OR - The effect size - For binary data, the odds ratio is displayed by default. You can change the metric by selecting metric from the Analysis menu.
  • lower - The lower bound of the confidence interval
  • upper - The upper bound of the confidence interval
