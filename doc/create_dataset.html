Creating a New Data Set

You can create a new data set in either the Welcome to OpenMeta dialog or the main Open Meta-Analyst window. If you are working in the main Open Meta-Analyst window, do the following steps. (Note: If you are working in the Welcome to Open Meta dialog, you can skip step 1.)

  1. Select new dataset from the Dataset menu.

  2. In the New Dataset dialog (or the Welcome to OpenMeta dialog), type the name of the data set:

  3. Select the type of data you are working with. In this example, the data type is binary.

    See Data Types for description of the data types available in Open Meta-Analyst.

  4. Select create new. This opens the add new outcome dialog shown below.

  5. In the dialog, type a name for the outcome of the test, and select the type of outcome, depending on the type of data you are analyzing.
  6. Click OK.

The Open Meta-Analyst window now displays a row of fields where you can enter data for a study. For binary data, the row appears as shown below.
