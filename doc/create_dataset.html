Creating a New Data Set

To create a new data set:

  1. Type the name of the data set:

  2. Select the type of data you are working with. In this example, the data type is binary.


  3. Select create new.

    This opens the dialog shown below.


  4. In the dialog, type a name for the outcome of the test, and select the type of outcome, Binary or Continuous, depending on the type of data you are analyzing. Then click OK.

The Open Meta-Analyst window now displays a row of fields where you can enter data. For binary data, the row appears as shown below.
