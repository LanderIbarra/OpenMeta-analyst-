Adding a covariate

To add a covariate to your data set,

  1. Select add covariate from the Dataset menu.
  2. In the add a new covariate dialog, type a name for the covariate in the covariate name field, as shown below.


  3. Select the type of covariate: Continuous or Factor. In this example, the covariate is continuous.
  4. Click OK.

The Open Meta-Analysis window now displays a new column for the covariate. You can enter data for the covariate by double-clicking in the column and typing values. The picture below shows the window with added data for the covariate.
