Starting Open Meta-Analyst

To start Open Meta-Analyst:

  1. Open the folder where you extracted the Open Meta-Analyst program files.
  2. In the "dist" subfolder, double-click the file

This opens the Open Meta-Analyst window, as shown below.

Note: You can create a shortcut to Open Meta-Analyst by dragging the OpenMetaAnalyst.exe icon onto the desktop.

In the Welcome to Open Meta dialog, you can create a new data set or open an existing data set. The next section shows how to create a new data set.
