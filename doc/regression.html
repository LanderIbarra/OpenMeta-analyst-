Performing a Meta-Regression

This section provides an example that illustrates how to perform a meta-regression. To try the example for yourself, open the data set meantime.oma, as shown below.

Then do the following steps:

  • Select meta-regression from the Analysis menu.
  • In the select covariates dialog, select the covariate or covariates that you want to perform the regression against.

  • Click OK.

This displays a regression summary and plot, as shown below.
