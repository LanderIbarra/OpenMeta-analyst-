Changing the Metric for the Effect Size

For two-arm studies with binary data, as shown in the preceding example, the default metric for the effect size is the odds ratio (OR). To change the metric:

  1. Select metric in the Analysis menu.
  2. Select two-arm.
  3. Select a metric from the pop-up menu, as shown below.

    When you select a metric, the effect sizes and confidence intervals in the data table are immediately updated.

    Note: If you select one-arm, only the first two data columns (Tx An and Tx AN) are included in the analysis. The Tx Bn and Tx BN columns are shaded grey.
