Opening an Existing Data Set

To open an existing data set:

  1. Select open from the File menu.
  2. Select the file BCG.oma.
  3. Click Open.

Open Meta-Analyst displays the data in a table as shown below.

Each row in the table contains the data from an individual study. In this example, the data is binary and the studies have two arms. The columns labeled "tx" display the numbers of events and subjects in the treatment and control groups:

  • tx An - Number of events in the treatment group
  • tx AN - Number of subjects in the treatment group
  • tx Bn - Number of events in the control group
  • tx BN - Number of subjects in the control group

You can select the studies to include in a meta-analysis using the check boxes in the "include" column.

The columns shaded yellow display the effect sizes, which are automatically calculated from the data:

  • OR - The effect size - For binary data, the odds ratio is displayed by default.
  • lower - The lower bound of the confidence interval
  • upper - The upper bound of the confidence interval
