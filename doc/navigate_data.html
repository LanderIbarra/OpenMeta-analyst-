Navigating a Data Set

A data set can contain multiple outcomes, follow-ups, and treatment groups. For instance, the example data set amino.oma contains two outcomes, as shown in the edit dataset dialog.

However, the Open Meta-Analyst window only displays one outcome at a time - the outcome you are currently working with. You can select which outcome to work with using the navigation arrows at the bottom of the data table.

The currently selected outcome is displayed to the right of outcome. Click the right or left arrow keys to advance to the next outcome

Similarly, you can use the navigation arrows to select which follow-up or pair of treatment groups to work with. Click the up-down arrows to cycle through the three categories:

  • Outcome
  • Follow-up
  • Treatment group

For example, click the up-down arrows until you reach Treatment group.

The currently selected pair of treatment groups is displayed at the top of the data window.

In this example, there is just one pair of treatment groups. However, the treatment groups can be displayed in two different orders: tx A - tx B or tx B - tx A. To switch the order, click the right navigation arrow. The result is shown below.

Note: If there are more than two treatment groups, the right-left arrows cycle though all possible ordered pairs of treatment groups.
