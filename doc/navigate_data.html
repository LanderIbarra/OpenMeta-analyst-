Navigating a Data Set

The Open Meta-Analyst window only displays one outcome, one follow-up, and one pair of treatment groups at a time. You can select which of these is displayed using the navigation controls at the bottom of the data table.

Click the up-down arrows to cycle through the three categories:

  • Outcome
  • Follow-up
  • Treatment group

Once you have selected a category, click the left-right arrows to cycle through all the subsets of data you have created for that category.

Note: If group is selected, the left-right arrows cycle through all possible ordered pairs of treatment groups.
