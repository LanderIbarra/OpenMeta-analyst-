Editing Components a Data Set

To edit the components of a data set, select edit on the Dataset menu. This opens the edit dataset dialog. For the example data set amino.oma, the dialog appears as shown below.

In the edit dataset dialog, you can add or delete studies, outcomes, follow-ups, treatment groups (study arms), and covariates. For example, to add a new treatment group:

  1. Click the plus button below the tx groups pane.

  2. In the add new tx group dialog, type a name for the treatment group.

  3. Click OK. The new outcome is displayed in the outcomes pane.

  4. In the edit dataset dialog, click OK.
