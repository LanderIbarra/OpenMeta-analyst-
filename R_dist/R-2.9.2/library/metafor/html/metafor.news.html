metafor.news {metafor}R Documentation

Read News File of the Metafor Package

Description

Read news file of the metafor-package package.

Usage

metafor.news(...)

Arguments

... arguments passed to file.show.

Details

The function just makes use of the file.show command to display the ‘NEWS’ file of the package.
